This directory contains scripts that are for Intel internal use only
and will not be released with public github

- run-with-arc.sh
  This is a helper script that acquires required arc resources.  If run
  on a non-ARC system it checks if the required resources are available.

ASE is Work in progress:
  The ASE directory contains script that will eventually support running tests
  with ASE. These are currently a work in progress and are not yet fully
  functional on generic systems. For now use existing ASE scripts as a fallback
  to run ASE.
